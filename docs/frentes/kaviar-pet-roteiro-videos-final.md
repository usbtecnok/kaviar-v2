## 3. Estrutura Prática de Gravação

### Equipamento mínimo:

| Item | Opção econômica | Opção melhor |
|------|----------------|-------------|
| Gravação de tela | OBS Studio (grátis) | Loom Pro |
| Slides | Canva (apresentação) | Google Slides |
| Narração | Microfone do celular (ambiente silencioso) | Microfone lapela USB (~R$50) |
| Edição | CapCut (grátis) | DaVinci Resolve (grátis, mais completo) |
| Música | Pixabay Audio (grátis, sem copyright) | Epidemic Sound (pago) |

### Fluxo de gravação:

```
1. Montar slides no Canva (usar templates SVG como base)
2. Ensaiar narração 1x (ler em voz alta, marcar pausas)
3. Abrir OBS Studio → captura de tela + microfone
4. Apresentar slides no Canva (modo apresentação)
5. Narrar slide a slide (pausar entre slides se necessário)
6. Exportar vídeo bruto (MP4, 1080p)
7. Editar no CapCut: cortar silêncios, adicionar música, transições
8. Exportar final: MP4, 1080p, 30fps
9. Upload YouTube (não-listado)
```

### Configurações de exportação:

| Parâmetro | Valor |
|-----------|-------|
| Resolução | 1920x1080 (Full HD) |
| FPS | 30 |
| Formato | MP4 (H.264) |
| Áudio | AAC, 128kbps |
| Thumbnail | 1280x720, com logo + título |

### Transições recomendadas:

- Entre slides: fade (0.3s) ou corte seco
- Não usar transições elaboradas (girar, zoom, etc)
- Manter ritmo constante

### Música de fundo:

- Instrumental, sem letra
- Volume baixo (20-30% do volume da voz)
- Estilo: corporativo/tecnologia/calmo
- Buscar em: Pixabay Audio → "corporate ambient" ou "technology calm"
