## 1. Criar Google Forms — Pré-Cadastro KAVIAR Pet

### Passo a passo

1. Abrir https://forms.google.com → **Formulário em branco**
2. Clicar no título e colar:


3. Clicar na descrição e colar:

```

Após o pré-cadastro, você será contatado via WhatsApp pela Central KAVIAR Pet para iniciar o processo de homologação:
```

## 2. Criar Google Forms — Questionário de Certificação

### Passo a passo

1. Abrir https://forms.google.com → **Formulário em branco**
2. Título:

```
KAVIAR Pet — Questionário de Homologação
```

3. Descrição:

```
Questionário obrigatório para certificação como Motorista KAVIAR Pet.

Responda com base no treinamento (Vídeo 1 e Vídeo 2).

• Nota mínima: 7/10 (70%)
• Você tem 2 tentativas
• Se reprovar, reassista os vídeos e tente novamente em 24h
```

4. **IMPORTANTE — Ativar modo teste:**
   - ⚙️ Configurações → Testes → "Transformar em teste" → **Ativar**
   - Liberar nota → "Imediatamente após o envio"
   - Mostrar respostas corretas → **Após o envio**

## 3. Criar Google Sheets — Central KAVIAR Pet

### Passo a passo

1. Abrir https://sheets.google.com → **Planilha em branco**
2. Renomear para:

```
KAVIAR Pet — Central Operacional (Piloto)
```

3. Criar 9 abas (clicar no "+" no rodapé):

| Aba | Nome exato |
|-----|-----------|
| 1 | Pré-cadastros |
| 2 | Homologação |
| 3 | Pagamentos |
| 4 | Treinamento |
| 5 | Fotos |
| 6 | Motoristas Aprovados |
| 7 | Corridas Piloto |
| 8 | Incidentes |
| 9 | Financeiro |

4. Para cada aba, colar os cabeçalhos na linha 1:

---

**Aba "Pré-cadastros"** — colar na linha 1 (A1 até O1):
```
Data	Nome	WhatsApp	E-mail	CPF	Cidade	Veículo	Ano	4 portas	Experiência pet	Disponibilidade 7d	Origem	Status	Responsável	Observações
```

---

**Aba "Homologação"** — colar na linha 1 (A1 até L1):
```
Nome	WhatsApp	Vídeos enviados	Vídeos assistidos	Questionário 1ª	Questionário 2ª	Status questionário	Fotos enviadas	Fotos aprovadas	Status geral	Data aprovação	Observações
```

---

**Aba "Pagamentos"** — colar na linha 1 (A1 até K1):
```
Data corrida	Motorista	Corrida ID	Valor total	Comissão motorista	Taxa KAVIAR	Método	Status	Data pagamento	Comprovante	Observações
```

---

**Aba "Treinamento"** — colar na linha 1 (A1 até H1):
```
Nome	WhatsApp	Data envio vídeos	Vídeo 1 assistido	Vídeo 2 assistido	Dúvidas	Status	Observações
```

---

**Aba "Fotos"** — colar na linha 1 (A1 até I1):

---

**Aba "Motoristas Aprovados"** — colar na linha 1 (A1 até K1):

5. **Formatação recomendada:**
   - Linha 1 de cada aba: **negrito** + fundo cinza claro
   - Congelar linha 1: Exibir → Congelar → 1 linha
   - Colunas de status: adicionar validação de dados (dropdown)

6. **Dropdowns de status (Dados → Validação de dados):**

   - Pré-cadastros coluna M: `Novo, Contatado, Em treinamento, Desistiu`
   - Homologação coluna G: `Pendente, Aprovado, Reprovado`
   - Homologação coluna J: `Pendente, Em andamento, Aprovado, Reprovado, Aguardando`
   - Pagamentos coluna H: `Pendente, Pago, Estornado`

   - Corridas Piloto coluna L: `Concluída, Cancelada, Incidente`

## 5. Observações para a Operadora

### Antes de iniciar

- Confirmar que todos os links estão preenchidos na tabela acima
- Testar cada formulário com envio de teste
- Verificar que o Sheets recebe as respostas do Forms
- Salvar mensagens prontas como respostas rápidas no WhatsApp

### Durante o piloto

- Toda comunicação passa pelo WhatsApp da Central
- Todo status é atualizado na planilha no mesmo dia
- Fotos são salvas no Drive com nome do motorista
- Dúvidas fora do protocolo → escalar para SUPER_ADMIN

### Horário de operação (piloto)

- Segunda a sexta, 9h-18h
- Fora do horário: mensagem automática do WhatsApp Business
- Emergências durante corrida: atender independente do horário
