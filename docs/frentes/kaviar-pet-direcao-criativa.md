## 1. Proposta Visual do Logo

### 1.1 Conceito

O logo KAVIAR Pet é uma **extensão controlada** da marca KAVIAR. Não é uma marca separada — é um módulo visual que herda a autoridade da marca-mãe e adiciona um elemento pet discreto e profissional.

**Metáfora visual:** como uma habilitação especial na carteira de motorista — o documento é o mesmo, mas há um selo adicional que indica capacidade extra.

### 1.2 Símbolo

| Opção | Descrição | Nível de discrição |
|-------|-----------|-------------------|
| **A (recomendada)** | Pata minimalista integrada à letra "K" ou ao "i" do KAVIAR | Máximo — quase imperceptível para quem não sabe |
| **B** | Pata pequena posicionada após "KAVIAR" como ponto/acento | Alto — visível mas não dominante |
| **C** | Silhueta de escudo com pata dentro, ao lado do logotipo | Médio — elemento separado mas contido |

**Direção recomendada: Opção A ou B.**

A pata deve ser:
- Geométrica (não orgânica/desenhada à mão)
- Monocromática (dourada ou branca, conforme fundo)
- Proporcional (não maior que uma letra do logotipo)
- Sem detalhes internos (sem "almofadinhas", sem unhas)

### 1.3 Tipografia

```
KAVIAR          → Montserrat Bold (ou Poppins SemiBold) — peso 700
Pet             → Mesmo font, peso 400 (light) ou 300
                   Posicionado: ao lado, abaixo ou como subscrito

Hierarquia:     KAVIAR é dominante. "Pet" é qualificador.
```

**Variações tipográficas:**

```
Opção 1:  KAVIAR Pet          (inline, Pet em peso leve)
Opção 2:  KAVIAR              (Pet abaixo, menor, centralizado)
              Pet
Opção 3:  KAVIAR | Pet        (separador vertical)
```

## 2. Proposta do Selo "Motorista Certificado"

### 2.1 Formato

**Escudo arredondado** (não círculo puro, não quadrado). Transmite proteção + autoridade.

```
        ╭──────────────╮
       │                │
       │   [ÍCONE]      │
       │                │
       │  KAVIAR Pet    │
       │  ───────────   │
       │  MOTORISTA     │
       │  CERTIFICADO   │
       │                │
       │   ★ 2026 ★     │
        ╰──────────────╯
```

### 2.2 Elementos e hierarquia

| Camada | Elemento | Posição |
|--------|----------|---------|
| 1 (fundo) | Escudo com borda dourada (`#B8942E`) | Contorno externo |
| 2 (centro) | Ícone: pata geométrica dentro de escudo menor | Centro superior |
| 3 (texto primário) | "KAVIAR Pet" | Abaixo do ícone |
| 4 (separador) | Linha fina dourada | Entre nome e título |
| 5 (texto secundário) | "MOTORISTA CERTIFICADO" | Abaixo do separador |
| 6 (rodapé) | Ano ou "★ APROVADO ★" | Base do escudo |

### 2.3 Cores do selo

| Variação | Fundo | Borda | Texto | Uso |
|----------|-------|-------|-------|-----|
| Premium (padrão) | `#1A1A1A` (preto) | `#B8942E` (dourado) | Branco + dourado | Certificado, destaque |
| Claro | `#FFFFFF` | `#B8942E` | Preto + dourado | Documentos, fundo claro |
| Monocromático | Transparente | Dourado | Dourado | Sobre fotos |

## 5. Ferramentas e Fluxo de Produção

### 5.1 Stack recomendada

| Ferramenta | Uso | Custo |
|-----------|-----|-------|
| **Canva Pro** | Logo, selo, slides, certificado, cards, PDF | R$35/mês |
| **Phosphor Icons** | Ícones consistentes (linha fina) | Grátis (phosphoricons.com) |
| **Feather Icons** | Alternativa de ícones | Grátis (feathericons.com) |
| **Google Fonts** | Montserrat + Inter (download) | Grátis |
| **Unsplash/Pexels** | Fotos de apoio (carro, pet, capa) | Grátis |
| **Figma** (opcional) | Se quiser mais controle vetorial | Grátis (plano starter) |

### 5.2 Fluxo de produção (ordem)

```
1. Criar paleta no Canva (Brand Kit)
   → Cores: #B8942E, #1A1A1A, #FFFFFF, #2D6B4F, #4A4A4A
   → Fontes: Montserrat Bold + Inter Regular

2. Criar logo KAVIAR Pet
   → Começar com texto "KAVIAR Pet" na tipografia definida
   → Adicionar elemento pata (opção A ou B)
   → Exportar variações (horizontal, compacto, mono)

3. Criar selo certificado
   → Template escudo no Canva
   → Adicionar elementos na hierarquia definida
   → Exportar: 1080x1080, 200x200, 64x64

4. Criar template de slide
   → Fundo escuro, header com logo, footer com texto
   → Criar 5 layouts-base (capa, título, lista, alerta, destaque)
   → Duplicar para os 44 slides (22+22)

5. Preencher slides com conteúdo
   → Usar roteiro já documentado
   → Adicionar ícones Phosphor/Feather
   → Revisar fluxo visual

6. Criar certificado digital
   → Template A4 landscape
   → Campos personalizáveis (nome, data, ID)

7. Criar cards de divulgação
   → 3-5 cards quadrados (1080x1080) para WhatsApp/Instagram
```

### 5.3 Tempo estimado

| Entregável | Tempo (Canva Pro, 1 pessoa) |
|-----------|---------------------------|
| Logo (todas variações) | 2-3 horas |
| Selo (todas variações) | 1-2 horas |
| Template de slides | 1-2 horas |
| 44 slides preenchidos | 4-6 horas |
| Certificado digital | 1 hora |
| Cards divulgação | 1-2 horas |
| **Total** | **10-16 horas (2-3 dias)** |

### 5.4 Bibliotecas de templates Canva (atalho)

Buscar no Canva por:
- "certificate dark gold" → base para certificado
- "presentation dark minimal" → base para slides
- "badge seal gold" → base para selo
- "logo minimal" → inspiração para logo

Adaptar com a paleta KAVIAR Pet (não usar as cores do template).
