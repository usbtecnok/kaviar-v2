## 1. Direção Visual

O KAVIAR Pet deve transmitir:

| Atributo | O que significa visualmente |
|----------|---------------------------|
| Segurança | Cores sólidas, formas estáveis, sem elementos frágeis |
| Premium | Tipografia elegante, espaço negativo, acabamento limpo |
| Operacional | Ícones funcionais, linguagem direta, sem excesso decorativo |
| Certificado | Selo, badge, marca de aprovação visível |
| Assistido | Presença humana implícita (central, suporte, acompanhamento) |

### O que KAVIAR Pet NÃO é visualmente:

- ❌ Infantilizado (sem patinhas coloridas, sem fontes arredondadas "fofas")
- ❌ Genérico (sem clip-art de cachorro sorrindo)
- ❌ Amador (sem cores neon, sem gradientes excessivos)
- ❌ Apenas "pet" (não é pet shop, não é veterinária)

### O que KAVIAR Pet É visualmente:

- ✅ Extensão da marca KAVIAR (mesma família visual)
- ✅ Operação profissional (como aviação, logística, saúde)
- ✅ Certificação (como selo de qualidade, ISO, habilitação)
- ✅ Confiança (tutor entrega o pet sabendo que há protocolo)

## 2. Paleta de Cores

### Cores primárias (herdadas do KAVIAR):

| Cor | Hex | Uso |
|-----|-----|-----|
| Dourado KAVIAR | `#B8942E` | Marca principal, selo, destaques |
| Preto KAVIAR | `#1A1A1A` | Textos, fundos premium |
| Branco | `#FFFFFF` | Fundos limpos, contraste |

### Cores secundárias (KAVIAR Pet):

| Cor | Hex | Uso | Motivo |
|-----|-----|-----|--------|
| Verde-escuro | `#2D6B4F` | Acento operacional, badges, status "aprovado" | Confiança, saúde, segurança |
| Cinza-grafite | `#4A4A4A` | Textos secundários, bordas | Sobriedade |
| Âmbar suave | `#D4A843` | Alertas, atenção, divergências | Extensão do dourado para warnings |

### Combinações recomendadas:

- **Selo aprovado:** Dourado + verde-escuro
- **Alerta/divergência:** Âmbar + preto
- **Central operacional:** Preto + branco + verde-escuro
- **Materiais para motorista:** Dourado + preto + branco
- **Materiais para tutor:** Verde-escuro + branco + dourado

## 3. Selo "Motorista KAVIAR Pet Aprovado"

### Conceito:

O selo é a marca de certificação do motorista. Deve funcionar como um badge profissional — similar a certificações de segurança, habilitações especiais ou selos de qualidade.

### Elementos do selo:

```
┌─────────────────────────────────┐
│                                 │
│     ╔═══════════════════╗       │
│     ║   KAVIAR Pet      ║       │
│     ║   ─────────────   ║       │
│     ║   MOTORISTA       ║       │
│     ║   CERTIFICADO     ║       │
│     ║                   ║       │
│     ║   [ícone: escudo  ║       │
│     ║    com pata]      ║       │
│     ║                   ║       │
│     ║   ★ APROVADO ★    ║       │
│     ╚═══════════════════╝       │
│                                 │
└─────────────────────────────────┘
```

### Diretrizes do selo:

- Formato: escudo ou círculo com borda dourada.
- Ícone central: pata estilizada dentro de escudo (segurança + pet).
- Texto: "KAVIAR Pet — Motorista Certificado" ou "KAVIAR Pet Aprovado".
- Cores: dourado + verde-escuro sobre fundo escuro, ou dourado + preto sobre fundo claro.
- Sem elementos infantis (sem carinhas de cachorro, sem ossinhos, sem corações).

### Variações:

| Variação | Uso |
|----------|-----|
| Selo completo (com texto) | Certificado digital, materiais impressos |
| Badge compacto (ícone) | App, WhatsApp, avatar |
| Selo horizontal | Assinatura de mensagem, rodapé |
| Selo monocromático | Documentos formais, termos |

## 5. Conceito Visual da Central KAVIAR Pet

A Central não é um "chat de suporte". É um centro de operações.

### Linguagem visual da Central:

- Dashboard escuro (fundo `#1A1A1A` ou `#2A2A2A`).
- Cards de corrida com status colorido (verde = OK, âmbar = atenção, vermelho = incidente).
- Timeline lateral mostrando etapas da corrida pet.
- Foto do pet visível no card da corrida.
- Indicadores: tempo desde chegada, checklist pendente, divergência reportada.

### Tom visual:

- Similar a: centro de controle de operações, painel de monitoramento.
- Não similar a: chat de atendimento, inbox de email, feed social.
