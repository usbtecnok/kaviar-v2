## 1. Checklist de Aprovação Admin

### Pré-cadastro

- [ ] Motorista preencheu formulário completo
- [ ] Telefone/WhatsApp válido
- [ ] Cidade atendida pelo KAVIAR
- [ ] É motorista KAVIAR ativo OU novo cadastro aceito
- [ ] Sem histórico de suspensão grave

### Adequação + Taxa de Homologação

- [ ] Motorista recebeu lista de itens obrigatórios
- [ ] Motorista confirmou que adquiriu os itens
- [ ] Taxa de Homologação KAVIAR Pet paga (Pix recebido — R$29-49)
- [ ] (Opcional) Se comprou Kit Oficial: pagamento confirmado + retirada/envio feito

### Treinamento

- [ ] Link dos 2 vídeos enviado via WhatsApp
- [ ] Motorista confirmou que assistiu ambos
- [ ] Questionário respondido
- [ ] Nota ≥ 7/10
- [ ] Se reprovado na 1ª tentativa: aguardou 24h e refez

### Fotos de comprovação

- [ ] **Foto 1:** Capa protetora instalada no banco traseiro (visível, bem presa)
- [ ] **Foto 2:** Kit/desinfetante visível no veículo (spray + pano + sacos)
- [ ] **Foto 3:** Banco traseiro preparado (capa + espaço limpo para animal)
- [ ] (Opcional) **Foto 4:** Selfie com kit instalado

### Validação das fotos

- [ ] Fotos são do veículo do motorista (não genéricas/internet)
- [ ] Capa cobre o banco traseiro adequadamente
- [ ] Produtos do kit são visíveis e identificáveis
- [ ] Veículo está em condições aceitáveis
- [ ] Se foto suspeita: pediu nova foto com ângulo diferente

### Aprovação final

- [ ] Todos os itens acima conferidos
- [ ] Motorista marcado como "KAVIAR Pet Aprovado" na planilha
- [ ] Mensagem de confirmação enviada via WhatsApp
- [ ] Data de aprovação registrada

## 5. Critérios de Suspensão do Selo

| Motivo | Ação |
|--------|------|
| 2+ reclamações graves de tutores | Suspensão temporária + conversa |
| Kit não mantido (fotos de revalidação reprovadas) | Suspensão até regularizar |
| Recusa de higienização comprovada | Suspensão temporária |
| Transporte sem capa protetora | Advertência → suspensão se reincidente |
| Cobrança direta ao tutor (fora do fluxo) | Suspensão imediata |
| Maus-tratos ao animal | Suspensão definitiva + desligamento |
