# Imagens para Landing de Turismo

## Estrutura de Pastas

```
frontend-app/public/
├── hero/
│   └── hero.jpg          (1920x1080 - Hero principal com Rio de Janeiro)
└── combos/
    ├── aeroporto.jpg     (800x600 - Aeroporto/Transfer)
    ├── citytour.jpg      (800x600 - Cristo/Pão de Açúcar)
    └── praias.jpg        (800x600 - Praias cariocas)
```

## Recomendações

### Hero (hero.jpg)
- Dimensões: 1920x1080px
- Sugestão: Vista panorâmica do Rio (Cristo, Pão de Açúcar, ou orla)
- Formato: JPG otimizado (< 500KB)

### Combos
- Dimensões: 800x600px cada
- Formato: JPG otimizado (< 300KB cada)

**aeroporto.jpg**: Aeroporto GaleãoSantos Dumont ou carro executivo
**citytour.jpg**: Cristo Redentor, Pão de Açúcar ou Centro Histórico
**praias.jpg**: Ipanema, Leblon ou Copacabana

## Placeholder Temporário

Enquanto não tiver as fotos, o CSS já está preparado com:
- Gradiente overlay para garantir legibilidade do texto
- Fallback para cor sólida caso a imagem não carregue
